What is Pando?

Pando is a powerful domain-specific AI prompt language designed for building sophisticated AI agents. With Pando, you can create, manage, and execute complex AI prompts with ease, streamlining your AI development workflow.

Pando allows you to define structured prompts that can include variables, conditions, and even nested prompts. This structured approach enables you to create more dynamic and context-aware AI interactions.

Intuitive Syntax

Write prompts in a clear, structured format that's easy to read and maintain. Pando's syntax is designed to be familiar to programmers while remaining accessible to non-technical users.

AI Agent Creation

Develop intelligent AI agents capable of performing complex tasks and reasoning. Pando's structure allows you to create agents with specific roles, knowledge bases, and decision-making capabilities.

Seamless Integration

Integrate Pando seamlessly with your existing development environment and workflows. Use Pando prompts in your applications, chatbots, or any system that interacts with AI models.

Extensible Framework

Extend Pando's capabilities with custom functions and integrations to suit your needs. Create reusable prompt components and build complex AI systems with ease.

Quick Start Tutorial

  1. Create a new Pando file:
    • Use the command palette (Ctrl+Shift+P or Cmd+Shift+P)
    • Search for "Pando: Create New File" and select it
    • Choose a name for your file (e.g., "my_first_prompt.pando")
  2. Write your first prompt: 
    prompt myFirstPrompt {
    "Hello! I'm an AI assistant created with Pando. 
    What would you like to know about Pando?"

    user_input = input()

    if user_input contains "syntax" {
        "Pando uses a simple, intuitive syntax. Here's a basic example:
        prompt examplePrompt {
            'This is a simple Pando prompt'
        }"
    } else if user_input contains "integration" {
        "Pando can be easily integrated into your existing projects.
        You can use Pando prompts in web applications, chatbots, 
        or any system that interacts with AI models."
    } else {
        "Pando is a powerful tool for creating AI prompts. 
        It allows you to structure your prompts with variables, 
        conditions, and even nested prompts. What specific 
        aspect would you like to know more about?"
    }
}
  3. Run your prompt:
    • Use the command palette again
    • Search for "Pando: Run Prompt" and select it
    • Your prompt will be executed, and you'll see the output in the terminal
  4. Interact with your AI:
    • The AI will greet you and ask what you'd like to know about Pando
    • Try asking about "syntax" or "integration" to see how the prompt responds differently
    • Experiment with different inputs to explore the prompt's behavior
